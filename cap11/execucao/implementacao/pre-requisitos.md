### Sistema operacional Linux 

1. Instale o _awscliv2_

```
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
```

```
unzip awscliv2.zip
```

```
sudo ./aws/install
```

2. Verifique se o _awscliv2_ foi instalado corretamente.

```
aws --version
```
